## Injections
- Supplying untrusted input
- Impact could be:
	- Loss of data
	- Corruption of information
	- Theft of information
	- DOS/DDOS

### Injection Types
- Structured Query Language (SQL)
- Lightweight directory access protocol (LDAP)
- Extensible Markup Language (XML)
- Dynamic Link Library (DLL)
- Command Injection
